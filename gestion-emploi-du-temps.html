Emplois du Temps

Configurez les horaires de l'établissement, détectez vos classes, puis générez automatiquement un emploi du temps sans conflit d'enseignant ni de section.

1. Configuration des horaires

Ces réglages définissent la grille de créneaux utilisée pour la génération.

Cochez, pour chaque jour, les périodes actives (Matin / Après-midi / Soirée). Par défaut : Lundi → Vendredi, Matin + Après-midi.

2. Classes / Sections

Une classe = un niveau + un groupe (ex : « 1ère année collège — Groupe A »).
